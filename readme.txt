A simple plugin to customize the default WordPress REST API prefix.

== Description ==

Enabling the ability to set your own WordPress REST API Prefix (/wp-json).

= Contribute to Custom REST API Prefix =
Development for this plugin happens in a public GitHub repository. If you have a suggestion, bug report, patch, or pull request, please feel free to [contribute here](https://github.com/alvindcaesar/custom-rest-api-prefix).

== Installation ==

1. Activate the plugin.
2. Go to Settings > Permalinks and set your custom prefix.
